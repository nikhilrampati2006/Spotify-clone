# 🎵 Spotify Clone

A responsive Spotify Web Player clone built using HTML and CSS.

## Features
- Responsive UI
- Sidebar navigation
- Music player section
- Album cards
- Font Awesome icons

## Technologies Used
- HTML5
- CSS3
- Flexbox
- Google Fonts
- Font Awesome
